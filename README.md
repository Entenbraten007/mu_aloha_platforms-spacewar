# UEFI for Nothing Phone (1) (spacewar)

EDK2-based UEFI port for the Nothing Phone (1) (Snapdragon 778G+).
This project is based on [mu_aloha_platforms](https://github.com/Project-Aloha/mu_aloha_platforms).

## Project Status
- **UEFI:** Boots
- **Display:** Not working (Stuck at splash screen/logo)
- **USB:** Not working
- **OS Booting:** Boots Android via LinuxLoader (when DefaultBDSBootApp is set to LinuxLoader)
- **Buttons:** Untested
